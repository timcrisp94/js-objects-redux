Iterating over an object's properties

We can iterate over an object's properties using a for...in loop

for (let key in game) {

console.log(`The value of the ${key} property is ${game[key]}`)

}

(Or a good old for loop - but for in works better)

Super Hot Object methods

Object.keys()

Object.keys() returns an array of a given object's own enumberable property names iterated in the same order that a normal loop would

const object1 = {

a: 'something',

b: 42,

c: false

}

console.log(Object.keys(object1))

//expect output: Array ['a','b', 'c']

Object.values()

Object.values() returns an array of a given object's own enumberable property values iterated in the same order that a normal loop would

const object1 = {

a: 'something',

b: 42,

c: false

}

console.log(Object.values(object1))

//expect output: Array ['something', 42, false]

Object.entries()

Object.entries() returns an array of the whole kit and caboodle

const object1 = {

a: 'something',

b: 42,

c: false

}

for (const [key, value] of Object.entries(object1))

console.log(`${key}: ${value}`)

//expect output: 'a: something' b: 42 c: false
